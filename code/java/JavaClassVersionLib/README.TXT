JavaClassVersionLib
===================

Related post:
http://illegalargumentexception.blogspot.com/2009/07/java-finding-class-versions.html
http://illegalargumentexception.blogspot.com/2009/11/java-application-to-check-binary-class.html

Demo library for reading class versions from Java binaries.

The jar contains a small GUI demo application which can be run with the
command:

    java -jar classver_1.1.jar

The code will run on Java 1.4 and above.
