JsfClientId
===========

Related post:
http://illegalargumentexception.blogspot.com/2009/10/jsf-working-with-component-identifiers.html

This project provides JSP and Facelet tag library functions for working with JSF components clientIds.

License: MIT (see LICENSE.TXT)


BUILDING
========

A copy of JSF 1.2 is required to build this project. JSF 1.2 is included in any Java EE 5
compliant server.

The Java source compliance is Java 5.


USAGE
=====

clientId_*.jar should be placed in WEB-INF/lib of your JSF web application. See
blog post for more information.
