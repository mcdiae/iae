Gurning Faces
=============

A JavaServer Faces library.

See doc/gurn.html for documentation.

Related blog post:
http://illegalargumentexception.blogspot.com/2011/11/jsp-arbitrary-attributes-on-jsf.html

Building
========

Use Apache Maven 2 to build the project.

Usage
=====

Add the target/gurn-version.jar to your J2EE 1.4 project's WEB-INF/lib directory.
