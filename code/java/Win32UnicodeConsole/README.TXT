Win32UnicodeConsole
===================

Related post:
http://illegalargumentexception.blogspot.com/2009/04/java-unicode-on-windows-command-line.html

Sample Java demo that uses JNA to call the Windows API to emit/read Unicode
from the console.

Set the console (cmd.exe) to a TrueType Unicode font
Use Ant to build (build.xml)
Run the demos using runtests.bat
