Runtime scripts
===============

Usage:

i) ensure JAVA_HOME is set
ii) use Apache Ant to run the build.xml in the root to download dependencies and build the project

1) startServer.bat    - starts OpenEJB in new, minimized console
2) deployEar.bat      - deploys the EJB to the OpenEJB container
3) runClient.bat      - runs the client application
4) undeployEar.bat    - removes the EJB from the container
5) stopServer.bat     - stops OpenEJB
