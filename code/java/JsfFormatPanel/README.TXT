JsfFormatPanel
==============

Related post:
http://illegalargumentexception.blogspot.com/2009/08/jsf-custom-format-panel-control-for.html

License: MIT


BUILDING
========

Run Ant on the provided build.xml to build the project.
Build-time dependencies will be automatically downloaded (Apache libs used).


USAGE
=====

Java 5 and JSF 1.2 required.

formatPanel_*.jar should be placed in WEB-INF/lib of your JSF web application. 
See blog post for more information.
